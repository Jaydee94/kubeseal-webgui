# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.x.x   | :white_check_mark: |
| 3.x.x   | :x:                |
| 2.x.x   | :x:                |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please use GitHub's private vulnerability reporting feature:
1. Go to the [Security tab](https://github.com/Jaydee94/kubeseal-webgui/security) of this repository.
2. Click **"Report a vulnerability"**.
3. Fill in the details of the vulnerability and submit the report.

The maintainers will review the report and respond as soon as possible. We appreciate your help in keeping this project secure.
